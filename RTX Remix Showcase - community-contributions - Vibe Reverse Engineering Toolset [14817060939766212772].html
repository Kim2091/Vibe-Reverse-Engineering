[Personal Project, NV not involved , use at your own risk] Just created a whole set of python tools for command line to use with Cursor primarily to let AI/LLMs inspect game binaries, extract data and do whatever you ask it. Mileage ofc may vary depending on model quality vs task complexity, but I managed to break NFSU2 and Morrowind frustum culling with very little effort/babysitting on the same day while building these tools https://github.com/Ekozmaster/Vibe-Reverse-Engineering What the LLM will be able to do (and you just watch, approve/reject, guide):
  • Attach to a live process for tracing, breakpoints, and memory inspection
  • D3D9 draw call profiling and caller sampling
  • Monitor memory writes with backtraces and count function hits over time
  • Instruction-level stepping and register/stack/backtrace capture at runtime
  • Generate .asi DLL patches from a declarative JSON spec
  • Disassemble and decompile (Ghidra-quality) any function in a PE binary
  • Feed a persistent knowledge base (structs, enums, signatures, function/constants names) into decompilation for readable C++ output
  • Dump and trace C++ vtables and indirect call sites
  • Resolve MSVC RTTI class hierarchies from vtables
  • Reconstruct struct layouts from memory access patterns
  • Map callers, callees, cross-references, and data references across an entire binary
  • Search strings, byte patterns, and imports with filtering
  • Read typed data (floats, pointers, structs) from raw binary offsets
  • Analyze minidumps (exception records, stack walks, module lists)
  • Works on both 32-bit and 64-bit PE executables and DLLs Usage:
  • Just clone or download the project as a zip from GitHub
  • Open the folder with Cursor, and follow the instructions on the README.md file/github project main page
  • Start a new chat, send a very descriptive prompt of the target game, where it is, what you want the AI to do, expected result, what mods/patches it already have, setup, etc.
  • The experience usually goes with the AI constantly requesting to run commands using the terminal and you having to accept them (It describes to you what the commands do, and you can ask too) Example of prompt in a new chat:
Read this project and help me disable Frustum culling in Morrowind (D:/Games/Morrowind/Morrowind.exe). I'm modding raytracing into it with RTX Remix and need stuff to show behind the camera in reflections/mirrors, specially lights. It is using MGEXE and MWSE with MCP code patcher. (edited)
